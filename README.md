# 🚢 Titanic Survival Analysis

End-to-end data analysis project on the Titanic dataset exploring 
survival patterns using Python, MySQL, and Excel.

## 📊 Tools & Technologies
- **Python** — Pandas, Matplotlib, Seaborn (EDA & Visualization)
- **MySQL** — Survival queries, GROUP BY, Subqueries
- **Excel** — Pivot tables, Survival summary

## 🔍 Key Questions Answered
- Survival rate by gender, class, and age group?
- Which passenger class had highest mortality?
- How did fare prices correlate with survival?

## 📁 Project Files
| File | Description |
|------|-------------|
| `Titanic_DA.ipynb` | Python EDA notebook |
| `titanic_mysql_queries.sql` | MySQL analysis queries |
| `titanic_survival_analysis.xlsx` | Excel summary & pivots |

## 📌 Key Findings
- Female passengers had ~74% survival rate vs ~19% males
- 1st class survived at 2x rate of 3rd class
- Children under 10 had significantly higher survival odds
